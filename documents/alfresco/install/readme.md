# Alfresco インストール  
  
## 1. Docker インストール  
```bash
sudo apt install docker.io
```
  
## 2. Docker Compose インストール  
```bash
sudo apt install docker-compose
```
  
## 3. Alfresco チェックアウト  
```bash
git clone https://github.com/Alfresco/acs-community-deployment.git
```
  
## 4.  
```bash
cd acs-community-deployment/
```
  
## 5.  
```bash
git checkout 2.2.0
```
  
## 6.  
```bash
cd docker-compose
```
  
## 7.  
```bash
docker-compose up
```
